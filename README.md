# 🚀 Космо-Таблица

Весёлый сайт для изучения **таблицы умножения без зубрёжки**.
Сделан для детей: крупные кнопки, звёзды, игры и — главное — **фокусы-секреты**,
которые объясняют, *как* запомнить, а не заставляют «долбить наизусть».

## ✨ Что внутри

- **🪄 Секреты-фокусы** — 10 приёмов запоминания:
  удвоение (×2, ×4), «часики» для ×5, **волшебные пальцы для ×9** (интерактивные!),
  «ноль в подарок» для ×10, перестановка множителей, квадраты, разбивка трудных примеров.
- **🎓 Умная тренировка** — сайт запоминает, что даётся тяжело, и подсовывает
  именно эти примеры. К каждому можно открыть подсказку-секрет.
- **🎮 Игры** — «Космо-битва» (успей ответить за 60 секунд) и «Найди пару».
- **🗺️ Карта прогресса** — сетка 10×10: видно, что уже «знаю», а что «учу».
- **⭐ Звёзды, уровни, серии** — мотивация продолжать.

Весь прогресс хранится **только в браузере ребёнка** (localStorage) — без регистрации и интернета.

## 🕹️ Как пользоваться

1. Открыть сайт (ссылка ниже).
2. Вписать своё имя вверху, выбрать зверька-талисмана (клик по нему).
3. Начать с раздела **«Секреты»** — посмотреть 2–3 фокуса.
4. Потом **«Учусь»** — тренироваться по 5–10 минут в день.
5. Заглядывать в **«Карту»**, чтобы видеть, как всё зеленеет 🌱

## 🧠 Методика (для родителей)

Таблица умножения кажется огромной — 100 примеров. Но на самом деле «трудных»
всего около 15. Всё остальное закрывается простыми приёмами:

- на 1 — число не меняется;
- на 10 — дописать ноль;
- на 2 и 4 — удвоить (и ещё раз удвоить);
- на 5 — «половинка и ноль» / счёт как минуты на часах;
- на 9 — фокус с пальцами;
- **перестановка множителей** режет таблицу пополам (3×8 = 8×3);
- трудные 6×7, 7×8 и т.п. — разбиваем на лёгкие: 6×7 = 6×5 + 6×2 = 42.

Ребёнок учит **приёмы**, а тренировка закрепляет их до автоматизма — без слёз и зубрёжки.

## 💻 Технически

Чистый HTML + CSS + JavaScript, без библиотек и сборки. Просто статические файлы:

- `index.html` — разметка
- `style.css` — оформление
- `app.js` — вся логика, методики и игры

### Запустить локально

Просто открыть `index.html` двойным кликом. Или через простой сервер:

```bash
npx serve .
```
